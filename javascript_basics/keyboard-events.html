JavaScript Keyboard Events

Keydown Event

Use Case: Capture and display the key being pressed down.

Keypress Event (Deprecated)

Note: Keypress is deprecated but still works in some browsers. Use Keydown or Keyup instead.

Keyup Event

Use Case: Detect when a key is released.

Focus Event

Use Case: Detect when an input gains focus.

Blur Event

Use Case: Detect when an input loses focus.

Input Event

Use Case: Detect changes in input value.
